## 🎯 En qué me especializo

- **Modelado dimensional** — star schema, arquitectura medallion, dim/fact con granularidad de negocio
- **Pipelines de datos** — ingesta, transformación y carga automatizada en SQL Server y Fabric Lakehouse
- **Dashboards operativos** — Power BI y Qlik Sense con métricas de impacto directo en operaciones
- **Machine Learning aplicado** — modelos predictivos con XGBoost, explicabilidad con SHAP
- **IA Generativa** — agentes de consulta en lenguaje natural sobre datos estructurados (Claude API + RAG)
